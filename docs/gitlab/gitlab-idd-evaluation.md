## 概要

`gitlab-idd` は GitLab イシューを非同期タスクキューとして使い、複数ノード間でタスクを分散実行する
イシュー駆動開発スキル。リクエスター（イシュー投稿・レビュー）とワーカー（実装・報告）の役割を分離し、
ポーリングデーモンによる自動起動もサポートする。
